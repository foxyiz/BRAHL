# pyUtils — Python utilities

**Formula:** `f(x, y) = z` — **`f/`** engine and **`x/`** actions are **not** in this folder.

Lives under **`FoXYiZ/pyUtils/`**. Run from **`KK/`**:

```powershell
python FoXYiZ\pyUtils\cleaner.py              # dry-run
python FoXYiZ\pyUtils\cleaner.py --apply      # archive ephemeral z/ → archive/cleanup/
python FoXYiZ\pyUtils\sync_personas.py
python FoXYiZ\pyUtils\reset_demo_data.py
python FoXYiZ\pyUtils\zBatchDash.py --name parallel_demo --logs z/parallel_demo_*.log
```

HTML reports from **pyUtils** stay here. FoXYiZ verify dashboards (`*_zDash.html`) land under **`FoXYiZ/z/<run>/`**.

Paths: `FOXYIZ_ROOT` = this package’s parent · `KK_ROOT` = FoXYiZ parent (`qoa_web`, `Docs`, `archive`).
